UAV Project Drawings Landing Page

UAV Drawing Packages

The typical UAV drawing package will contain a facilities section and a schematics section. You are tasked with creating the schematics. The schematics are based on consultant or sales one lines in conjunction with the finalized bill of materials.

The project engineering team has a dedicated documentation department resource that is solely allocated to schematics. This individual should be utilized as often as possible for the process of creating the baseline schematics. This is to free up the project engineers time to scrub the bill of materials while at the same time beginning to take into consideration the Theory of Operation and how the system needs to operate to meet the CLIENTS OBJECTIVES.

 

Schematics For Internal Review

Drawings completed and  that are ready for internal review are sent to the Engineering Department group email address so that one of the team members can pick up the review. The Director of Engineering will always be the ultimately responsible for reviewing UAV schematics drawings and Full UAV drawing packages. When sending the email for review please include the following

All completed Schematics  This should include

  • 100% IFC or later issuance of Facilities
  • Includes all facilities & All the schematics for every system
  • All the pull schedules
  • The full UAV Drawing Package should be set up in a BlueBeam session
  • BOM for procurement
  • All the Digital Signal Processor Site Files
  • The Theory of Operation

 

When sending the email please include the following information

  • Drive location of the drawings
  • BlueBeam Session already set up with authorization for review and editing
  • Review Completion Need by Date

 

Issuing Fully Approved UAV Drawing

Once the drawing package is fully reviewed and approved an email notification should be sent and should include the following individuals / email groups

  • UAVDAILYUPDATES@usus.net
  • Project Manager
  • Programmer

 

UAV Drawing Packages Check List

The typical UAV drawing package will contain many key components that were previously discussed. Below is the thought process and mental checklist that is gone through whenever schematics are reviewed. By the time the process gets to the drawing review you should have already considered every aspect of the project.

 

General

As you scrub the bill of materials you should be already completing your own first round of drawing checklist/review. Meaning you can not scrub a BOM without drawing the system. As you draw the system schematics you should ensure that each portion of the B.O.M scrub is practiced in the drawing creation and review process.

 

Additional Items to Consider

Title Blocks

Does each page have the correct title block information.

Connection key

Does the connection key show the termination pin outs for every connection used on that particular schematics?

Number Legend

Does the number legend show every cable number used on that particular schematic?

Terminal Strip Legend

Does the terminal strip legend show every terminal strip used on that particular schematic?

Location Markers

Do the schematic facilities location markers match the location marker nomenclature on the facilities drawings?
