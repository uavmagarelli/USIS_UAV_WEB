Project Engineer Job Closing Checklist

ALL project drawings are to have a Final As-Built Revision badge.  This includes the following

  • Architectural(s)
  • Schematics

 

A full set of approved, final drawings are to be printed to PDF and the final copy is to be saved to: Z:\Project Folder\Closeout\Closeout Disc for Client\Final As-Builts (PDF's).

 

If there are any hand-written sketches, notes, etc. remaining that are pertinent to the project, scan them and save to:

Z:\Project Folder\Engineering\Documentation\Scanned Notes

 

Delete ALL obsolete DSP files.  All final DSP files should have the following in the file name:

[Job#_Room # & Description_Engineer's Initials_Date_FINAL] ie: 123456_USIS Boardroom 1234_AM_03.05.12.

 

Update the Z:\Project Folder\Control\Control System Programming folder with all up-to-date files from the control systems programmer.  Create a text file in that folder indicating what company and programmer wrote the codes.

Confirm the uncompiled code files and system manuals have been saved to:

Z:\Project Folder\Closeout\Closeout Disc for Client\Control System (Uncompiled Final Revisions).

 

Delete all unused directories and documents.

 

UPGRADES ONLY: In original project root directory create folder "_UPGRADES."  In this folder create a text file that says; Upgraded under job number xxxxxx.

 

Confirm e-mail PST has been saved to Z:\Project Folder\Closeout\E-mail\Engineering.
