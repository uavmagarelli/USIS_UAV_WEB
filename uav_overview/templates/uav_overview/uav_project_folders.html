UAV Project Folder Landing Page

Every Audio Visual project starts with a default project folder stored on a centralized server. The folder is equipped with subfolders and base line documents to help each team sub-faction to complete their tasks while maintaining uniformed documentation.

 

UAV Drive—Job Folder Template

Close Out

This folder is where all the finalized documents end up for the client to receive as part of the standard USIS AV close out package. There are documents that the Project Engineer is responsible for procuring and placing in the correct folder. This is discussed later as part of the Project Engineer Close Out Procedure section.

CTRL = Control Folder (also audio)

This folder contains the last revision of the uploaded [compiled] code from onsite. The main folder includes a sub folder for Control Code and a sub folder for the Digital Signal Processors site files.

 
 

ENG = Engineering

This folder contains all the autoCAD, REVIT and PDF files related to Audio Visual Drawing Package(s). There are preloaded subfolders ready to go to help organize all the received so that each project stores files in a similar fashion. There are documents in this folder that should be there when you are assigned projects as well as documents you will be responsible for creating. This is discussed later as part of the Audio Visual Drawing Package Creation Prcess.

Order Entry

This folder is where all the starting documents can be found when you begin your Proejct Engineering Processes. The folder is already populated with the Finalized Bill of Materials, Finalized Scope of Work, all the Vendor Quotes and all other sales supplied supporting documents.

Pics = Pictures

This is where all the pictures taken on the project are stored for everyone's access.  The pictures taken and their reasoning vary drastically so each tam member is responsible for properly labeling subfolders and pictures so that the rest of the team can understand exactly what they are looking at.

PM = Project Manager

This folder is where the assigned Project Manager stores all their supporting documents. In this folder is where you can find the Project Schedule that the team agrees to work by. All the documents in this folder are generated by others.

Service

This folder is where the Service Department stores any of their project specific supporting documents. All the documents in this folder are generated by others.

Travel

This is where the team stores all the travel related information for all team member for that particular project. All the documents in this folder are generated by others.
