UAV Server & Drive Landing Page

Audio Visual—Finding The Files

Every Audio Visual project starts with a default project folder stored on a centralized server. The folder is equipped with subfolders and base line documents to help each team sub-faction to complete their tasks while maintaining uniformed documentation.

UAV Drive—Server Dedicated to Audio Visual Department

 Base Directory

All of the Audio Visual departments documents are stored on the UAV Server drive. This will be the main starting point of every project.

Jobs Folder

Every project that UAV works on is set up with a standardized project folder.  The folder breaks down projects based on Active, Completed or if they are in the design phase.

Jobs In Progress

Every active project you will work on will be located in the Job In Progress folder. These folders are automatically created when a project is awarded. When a project is assigned to you, you will receive a link to the newly created job folder.

NOTE:

 Jobs Closed

After a project is completed and all as built documentation has been sent and the warranty starts on the install the job is moved into the Jobs Closed Folder. The folder/file migration of the folder and documents from In Progress to Closed is done by others.
