UAV Engineering Drive Landing Page

The Engineering drive is where you can find many templates, calculators and various technical reference guides. Below are only some of the bullet point folders

 

UAV Engineering Drive

ACAD Blocks

In this folder you will find all the necessary files for creating a full UAV-Drawing package. This includes base files, blocks, details and config files.

 

Project File Structure Template

This is the base template used to create all the active jobs folders.

 
 

Training

This folder contains the shared documentation we receive when we go to various vendor trainings. This folder should be a treasure trove of technical information. This is not a depository for easily finable user manuals.

 

UAV project Engineer Documents

This folder is where all the starting documents can be found when you begin your Project Engineering Processes. The various files include template BOMS, TOOs, DSP files as well as work books and calculators to help on each assignment and task you may need to accomplish during the project life cycle.

 

USIS LAN Info

Here is where you will find the different LAN set ups needed when working in Pearl River, the PR shop and when testing systems in QC
