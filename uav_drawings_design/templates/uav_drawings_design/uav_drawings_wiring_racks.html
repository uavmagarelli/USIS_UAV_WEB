The Numbering Scheme—When Multiple Racks Are In The Same System

A key stipulation to the standardizations is of the numbering conventions  is when there are multiple racks in the same Audio Visual system. If there is only one(1) rack in the system then all the cables inside the rack would start with a 100 numbering. For example the first line level audio cable would be L1001.

The difference comes when there are multiple racks in the same Audio Visual system. In these cases the internal rack cable numbers will start with a prefix associated to their rack number. This means cables in rack number one(1) will have cables starting with a 100 number. Rack number two(2) will have cables starting with a 200 number.
