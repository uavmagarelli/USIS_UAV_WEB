Although there are numerous icons associated with the full AV Drawing package there are a few main staples that are seen often. These icons are associated to the mounted methodology of any particular device or supporting structure. 
 

Audio Visual Mounting Icons

Floor Box

A Square box represents the a floor core or floor box location. If a device is mounted under a conference table and is routed through a floor core or under floor cable pathway the square icon would be the appropriate representation.

 

Wall Mount

A circle with a vertical post represents a wall mounted device. Wall plates and Device in-wall storage boxes would be appropriate uses for this icon.

 

Rack Locations

A circle with a vertical post also represents a floor standing rack.  If the rack location.

 

Ceiling Locations

A circle is the indicator for any ceiling mounted device. Certain antennae's and the majority of speakers would be good examples of when to utilize this icon.

Dividing an Icon
 

When there are multiple locations on the facilities drawing sharing the exact same call out icon and nomenclature that tie back to the same head end location splitting the icon helps differentiate the locations. The top portion represents the way the facilities set show the location. The bottom portion represents the facilities room number of that particular location.
